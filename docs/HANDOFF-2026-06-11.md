## 3. Azioni titolare (per attivare la Parte C completa)

1. **Deploy Edge Function** (Supabase CLI):
   ```bash
   supabase functions deploy telegram-assistant --no-verify-jwt
   supabase functions deploy owner-morning-digest
   supabase functions deploy owner-evening-brief
   ```
2. **Schedule cron**:
   ```bash
   supabase functions schedule owner-morning-digest "0 8 * * *"
   supabase functions schedule owner-evening-brief "0 20 * * *"
   ```
3. **Attiva le skill** in `/admin/funzionalita`: `telegram_assistant`, `owner_morning_digest`, `owner_daily_brief`, `auto_clock_in`, `expense_tracking`, `stock_consumption`.
4. **(Sicurezza, consigliato)** Rigenera il `TELEGRAM_BOT_TOKEN` con BotFather (`/revoke` → nuovo token), aggiorna il secret Supabase e rifai `setWebhook` col `secret_token`. (Il bot funziona anche col token attuale.)
5. **Verifica end-to-end**: scrivi al bot "speso 30€ in detergenti", "ho finito una cera usandola in salone", "chiudi la giornata" → controlla `expenses`, `stock_movements`, `daily_brief`.
